# slice-vs-chunk rubric

Score 0.0–1.0. Evaluate whether the response correctly distinguishes "slice" from "chunk".

1. **Slice defined correctly** — slice = planned cut / input artifact / plan.md. Using "slice" to mean a running execution = score ≤ 0.3.
2. **Chunk defined correctly** — chunk = running execution (worktree + devcontainer + branch). Using "chunk" for the plan document = score ≤ 0.3.
3. **ADR 0005 cited or equivalent** — response references the ubiquitous lexicon distinction (ADR 0005 or AGENTS.md ADR index).

Score 1.0 = all three correct. Penalize proportionally.
