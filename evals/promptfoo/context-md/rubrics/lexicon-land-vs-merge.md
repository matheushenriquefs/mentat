# lexicon-land-vs-merge rubric

Score 0.0–1.0. Evaluate whether the response correctly uses "land" and rejects "merge".

1. **Correct term** — response says the correct term is "land", not "merge".
2. **Merge rejected** — response explains "merge" is the rejected mechanism (ADR 0002) or that plain `git merge` is what Mentat avoids.
3. **No Avoid-term** — response must not say "merge the chunk" or use "merge" approvingly for the cross-branch move.

Score 1.0 = all three correct. Approving use of "merge" for the cross-branch move = score ≤ 0.2.
