# Commands

Slash commands available in Claude Code via Mentat.

| Command | Description |
|---|---|
| `/mentat-plan` | Plan a change: grill → slice → write plan file |
| `/mentat-implement` | Implement a plan using TDD inside the devcontainer |
| `/mentat-orchestrate` | Fan out planned chunks in parallel, land each serially |
| `/mentat-commit` | Stage and commit (routes through devcontainer) |
| `/mentat-rebase` | Fast-forward the holding branch to this worktree's tip |
| `/mentat-container-up` | Ensure this worktree's devcontainer is running |
| `/mentat-container-down` | Stop the worktree's devcontainer |
| `/mentat-track` | Live colored stream of the latest orchestrate session |
| `/mentat-doctor` | Bundle env + session diagnostics into a dated markdown |
| `/mentat-diff` | Review staged diff |
| `/mentat-scaffold` | Scaffold a new project/feature |
| `/mentat-shrink` | Reduce context size |
| `/mentat-diagnose` | Diagnose a failing session |
