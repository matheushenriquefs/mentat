# lexicon-slice rubric

Score 0.0–1.0. Evaluate whether the response correctly defines "slice" vs "chunk".

1. **Correct correction** — response says "slice" is wrong for a running worktree; a running worktree is a "chunk".
2. **Slice defined accurately** — slice = planned cut / input artifact. _Not_ the running execution.
3. **No Avoid-term used** — response must not say "the running slice" or use "slice" to mean execution.

Score 1.0 = all three correct. Using "slice" for a running execution in any positive sense = score ≤ 0.2.
